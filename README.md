# NIFTY & BANKNIFTY GEX Analyzer

A Streamlit-based web application for real-time Gamma Exposure (GEX) analysis of Indian indices (NIFTY and BANKNIFTY).

## Features

- Real-time GEX calculation using NSE option chain data
- Interactive visualizations using Plotly
- Detailed strike-wise analysis
- Risk assessment and alerting
- Configurable parameters for different analysis types
